About The Mastermind Five-Guess Algorithm

Donald E. Knuth

An algorithm was devised in the late 1970s which allows a player to decipher the code within five guesses, as long as the following conditions are set:

  • There are four balls in the solution
  • The number of colours available is six
  • Colours are repeatable within the solution

The first step is to guess a sequence of 1122, where each integer 1-6 represents a colour. The algorithm continues by choosing a sequence that could eliminate the highest number of possible solutions each time. The number of possible solutions to this particular set-up is 64 = 1296!

That would take you quite a while to work through on paper. It would be almost impossible to calculate and keep track of in your head. Therefore, this algorithm is done far quicker by machines - if coded correctly to do so!
